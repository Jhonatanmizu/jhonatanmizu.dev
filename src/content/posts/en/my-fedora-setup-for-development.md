# 🐧 My Fedora Setup for Development

### Based on the project: Fedora Mizu Setup Dev

If you use **Fedora Linux** for development, you know that setting everything up manually can be time-consuming. With that in mind, I created the project:

🔗 https://github.com/Jhonatanmizu/fedora-mizu-setup-dev

The goal is simple: **automate my complete development environment with a single script**, ensuring productivity, standardization, and speed whenever I need to reinstall the system.
